# Fellowship Tracker v1.0.0 🎉

The first official release of Fellowship Tracker - a desktop app for tracking your dungeon progress in Coffee Stain's Fellowship!

---

## 📥 Download

**[Fellowship-Tracker-Setup.exe](https://github.com/mosshansson/fellowship-tracker/releases/download/v1.0.0/Fellowship-Tracker-Setup.exe)** (~150MB)

### Installation:
1. Download the installer
2. Run `Fellowship-Tracker-Setup.exe`
3. Follow the installation wizard
4. Launch and enjoy!

---

## ✨ Features

### 🎮 Track Your Progress
- **8 Regular Dungeons** - Track league, difficulty (1-6), time, and rating
- **4 Capstone Dungeons** - Simple completed tracking
- **5 League Tiers** - Contender → Adept → Champion → Paragon → Eternal

### 👤 Character Management
- **Multiple Characters** - Create characters for all 11 heroes
- **Auto-Calculated Rating** - Total rating sums all dungeon ratings
- **Account Name** - One name for all your characters

### 💾 Data Safety
- **File-Based Storage** - Your data saved to `fellowship-data.json`
- **Update-Proof** - Data survives app updates
- **Easy Backup** - Just copy the JSON file

### 🎨 Clean Interface
- **Game-Accurate Colors** - Gold and teal theme
- **Quick Overview** - Perfect for queue decisions
- **No Terminal Window** - Clean launcher included

---

## 🚀 Quick Start

1. **Install** the app
2. **Enter your account name** (e.g., "Oaz")
3. **Add a character** (select your hero)
4. **Click the character** to start tracking
5. **Click any dungeon** to add progress
6. Your **total rating auto-calculates!**

## 💾 Data Location

Your progress saves to:
```
C:\Users\[USERNAME]\AppData\Local\Programs\fellowship-tracker\fellowship-data.json
```

**Backup this file to preserve your progress!**
